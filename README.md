1) What is the difference between null and undefined?
- null holo ekta value jeta empty value and undefined holo ekta value jeta assign kora hoyni.


2) What is the use of the map() function in JavaScript? How is it different from forEach()?
- map() function holo ekta array method jeta array er prottek element er jonno ekta function call kore and ekta new array return kore. forEach() function holo ekta array method jeta array er prottek element er jonno ekta function call kore but ekta new array return kore na.


3) What is the difference between == and ===?
- == holo ekta comparison operator jeta value compare kore and === holo ekta comparison operator jeta value and type both compare kore.


4) What is the significance of async/await in fetching API data?
- async/await holo ekta syntax jeta asynchronous code ke synchronous code er moto likhte help kore. eta API theke data fetch korar jonno use hoy.

5) Explain the concept of Scope in JavaScript (Global, Function, Block).
- Scope holo ekta variable er accessibility. eta define kore je ekta variable kothay access kora jabe. 
ok Global Scope holo ekta variable er accessibility jeta code er kothay access kora jabe. 
ok, Function Scope holo ekta variable er accessibility jeta function er moddhe access kora jabe. 
ok Block Scope holo ekta variable er accessibility jeta block er moddhe access kora jabe.
